
ls :
        - Trié ordre ascii
        - Sur une ligne
        - separé par un espace

ls -r :
        PAREIL mais ordre ascii inversé        

ls -a :
        - Trié ordre ascii
        - Sur une ligne
        - separé par un espace
        - Commence par . et ..

ls -R :
        - premiere ligne = path du dossier (commence par .)
        - seconde ligne = listing comme ls
        - si autre dossier, alors on saute une ligne.

ls -l :
        - Trié ordre ascii
        - premiere ligne = total + nombre
        - un file/dir par ligne
        - droits users | nb | mqwa | mqwa | 1046 | Apr 21 08:20 | Makefile

ls file1 :
        - On affiche en prmeier sur une ligne (sans -l) tous les files en ordre ascci present dans les args
        - Pour chaque dossier, saut de ligne + 1ere ligne path et ensuite listing
